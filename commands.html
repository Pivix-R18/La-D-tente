Voici les différentes commandes mises à votre disposition

/msg [Joueur] [message] Pour envoyer un message privé à quelqu'un

/reply [message] Pour répondre à un message privé

/ignore Pour voir la liste des joueurs ignorés

/ignore [player] Pour ajouter/supprimé un joueur ignoré (permet de ne plus recevoir ces messages (dans le chat global ou en privé)

/ping Pour connaitre son ping

/sit Pour s'assoir

/list Pour voir la liste des joueurs connectés

/playtime [player] Savoir combien de temps un joueur ou soi-même à jouer sur le serveur.

/options Permet de paramétrer quelques options comme le fait d'éditer les panneaux avec shift+click ou le son des tags (@LePseudo dans le chat)

En visant un cadre /cmi itemframe invisible Permet de rendre un cadre invisible

En visant un cadre /cmi itemframe invulnerable Permet de rendre un cadre invulnerable

Autre possibilité

Double click sur un escalier, une dalle ou un tapis permet de vous assoir dessus (Shift pour se relever)

Shift+click-droit sur un panneau déjà posé permet de le modifier

Shift+click-droit sur un porte armure permet de le customiser

Shift+click-droit sur un tableau posé + Mollet vers le bas où vers le haut permet de changer de tableau (Click droit sur n'importe quel bloc pour annuler)
